## Description

Shogun Discord Theme is a **Shogun Raiden-inspired UI theme** for **BetterDiscord**.  
It features:

- Neon-inspired color palette
- Custom background image
- Transparent chat and member layers
- Rounded avatars and subtle highlights
- Neon-style scrollbars

This theme is based on the **Cyberpunk 2077 theme** by [patrykbielanin](https://patrykbielanin.github.io/discord-cyberpunk2077-theme/base.css).

---

## Installation

1. Install **BetterDiscord** if you haven’t already.
2. Download the `Shogunstyle.theme` file from this repository.
3. Open Discord → User Settings → **Themes** → **Open Theme Folder**.
4. Move the downloaded theme file into the theme folder.
5. Back in Discord, toggle the theme on.
